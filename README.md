## Introduction

CQA website like Quora or Stackoverflow. Users can sign in. Ask Questions, answer others questions.

## Running the code


    - Run as:

### client side (PORT 3000)
    $ cd frontend
    $ npm install
    $ npm start

### server side (PORT 3001)
    $ cd backend
    $ npm install
    $ node index


### postgresql database server runs on PORT 5432
